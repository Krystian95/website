Jolie is a new programming language which addresses distributed application programming. It is a full fledged service oriented language and it provides both basic primitives for achieving communication among different services and composition mechanisms for componing existing services into new ones. Programming with Jolie is an exiting experience which will transform the way you think a distributed application!
