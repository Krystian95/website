Basic data types

> Go on top

JOLIE supports six basic data types:
  • int: integers;
  • long: longintegers;
  • double: double-precision floats;
  • string: strings;
  • bool: booleans;
  • byte arrays.
Byte arrays can not be created directly by the programmer, but they are supported in case a service returns you one and you need to pass it to another service.

Vectors

> Go on top

Vectors (or arrays) are accessed by using the [] operator as in many other languages. Example:
Every JOLIE variable is a vector. JOLIE handles vector creation, packing and resizing automatically. You do not need to declare vectors. This is very handy when dealing with complex data, and JOLIE hides this mechanism when the programmer does not need a vector.
