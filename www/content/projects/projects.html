Projects involving Jolie:

Chor
Highlights:
  • a new programming language based on choreographies that supports code generation of Jolie services.

Chor is a new programming language for communicating systems, based on choreographies. Choreographies are global descriptions of system implementations, where the behaviour of all entities in a system is given in a single program. Choreographies enhance the quality of software, as they represent "formal blueprints" of how communicating systems should behave and offer a concise view of the message flows enacted by a system. In Chor, a choreography can then be projected (or compiled, if you like) onto a set of separate programs that can be run in a distributed system.

Project Website: http://www.chor-lang.org/
Jeye
Highlights:
  • A web designer for Jolie services.

jEye is web application which allows for the design of Jolie workflow by exploiting a web GUI developed in GWT. The server side of jEye is a Service Oriented Architecture fully implemented with Jolie. In particular, we exploit service recursion for developing the Jolie parser which converts the wirkflow definition provided by the web GUI into a real and executable Jolie service. At the present jEys is a prototype.

Project Website: http://jeye.sourceforge.net/
Joliepse
Highlights:
  • An IDE for Jolie powered by Xtext framework.

Joliepse is an Integrated Development Environment (IDE) for the JOLIE programming language, powered by the Xtext framework. Joliepse provides code templating, code completion, syntax coloring, navigation, reference finding, outline views and syntax checking / error markers.

Project Website: http://www.jolie-lang.org/joliepse.php
JoRBA
Highlights:
  • An adaptation framework written in Jolie.

JoRBA (Jolie Rule-Based Adaptation framework), is a proof-of-concept framework implementation of adaptation mechanisms written Jolie.

Project Website: http://www.jolie-lang.org/examples/tgc10/JoRBAv0.1.zip
Leonardo
Highlights:
  • Tiny yet powerful Web Server written in pure Jolie.

Leonardo is very flexible, and can scale from serving simple static HTML content to supporting powerful dynamic web applications. Leonardo is particularly suited to integrate Web Development with Service-Oriented Architectures (SOAs)

Project Website: http://sourceforge.net/projects/leonardo
PaaSSOA
Highlights:
  • A Cloud Computing platform for the deployment of SOAs, developed in Jolie.

PaaSSOA is a Service Oriented PaaS framework for deploying service oriented architectures in a Cloud Envirnoment. It is fully implemented in Jolie and it provides a web GUI developed in GWT for interacting with the system anche checking the status of all the deployed services. At the present PaaSSOA is a prototype.

Project Website: http://sourceforge.net/projects/paassoa
