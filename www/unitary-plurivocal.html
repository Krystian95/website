The Unitary and Plurivocal Programming Language

UNITARY

A single tool to manage the problems of digital integration in an organic way

The ownership of being unitary allows Jolie to be able to face all the problems of digital integration with a single mode, whether technological changes or approach or planning.

PLURIVOCAL

A tool able to dialogue with different applications using different protocols.

The property of being plurivocal allows the language to dialogue using different protocols and formats. In this way Jolie can adapt its self to the type of communication to be used according to the application with which it interacts.

UNITARY and PLURIVOCAL

A single means of problem solving that can dialogue with a multiplicity of existing realities

Being at once unitary and plurivocal makes Jolie, in its own way, unique and special. In fact, it allows both to design an integrated system in a uniform way, and to be able to dialogue with existing components and new ones that will arrive in the future.
