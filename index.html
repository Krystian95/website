For geeks

Create your own jolie scripts for connecting easily different applications and web services. Use the protocol you need: http, https, soap or develop your own custom protocol!

For web architects

Try to use the easy jolie web server leonardo and customize the server side of your web application! Use GWT library or JSON/RPC protocol for allowing interaction between your web app and leonardo!

For engineers

Exploit the service composition mechanisms supplied by Jolie (redirection, aggregation and smart aggregation) for building high valued distributed systems. Enjoy the embedding feature for customizing the topology of your system.

For academics

Investigate new language featires and theorems by starting from the Jolie formal model SOCK. SOCK is a process calculus which defines the semantics of Jolie. Please, consult our publications for details.
