PaaSSOA

A Cloud Computing platform for the deployment of SOAs, developed in Jolie.

PaaSSOA is a Service Oriented PaaS framework for deploying service oriented architectures in a Cloud Envirnoment. It is fully implemented in Jolie and it provides a web GUI developed in GWT for interacting with the system, checking the status of all the deployed services. At the present PaaSSOA is a prototype.

Project Website: http://sourceforge.net/projects/paassoa
